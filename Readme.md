# Smart IoT Dashboard

Real-time IoT monitoring dashboard using:

- NodeMCU ESP8266
- MQTT Protocol
- Mosquitto Broker
- Python MQTT Listener
- Streamlit Dashboard
- ThingSpeak Cloud Integration

## Features

- Live temperature monitoring
- Live humidity monitoring
- MQTT communication
- Real-time dashboard
- CSV logging
- Cloud upload to ThingSpeak
- Device status monitoring

## Tech Stack

- Python
- Streamlit
- MQTT
- Mosquitto
- ESP8266
- Arduino IDE
